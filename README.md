# Project Setup

```sh
npm install
```

## 📁 1. Clonar el repositorio

```bash
git clone git@github.com:PedroCM101/front.git
cd front
```

## ⚙️ 2. Copiar el archivo de entorno

```bash
cp .env.example .env
```

## 📦 3. Instalar dependencias

```bash
npm install
```

## 🚀 4. Levantar el servidor

```bash
npm run dev
```
